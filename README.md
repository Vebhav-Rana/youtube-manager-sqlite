# youtube-manager-sqlite
Python CLI project to manage YouTube videos using SQLite.

This project is a command-line YouTube manager built using Python and SQLite.
It stores video information permanently using a database.

## Features
- Add videos
- Update video details
- Delete videos
- View all videos from database

## Tools Used
- Python
- SQLite

## What I Learned
- Using SQLite with Python
- Writing SQL queries (CRUD)
- Database-based data persistence
